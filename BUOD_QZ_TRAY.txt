┌──────────────────────────────────────────────────────────────────┐
│                  SUSUNOD NA GAGAWIN                              │
└──────────────────────────────────────────────────────────────────┘

HAKBANG 1: I-download ang QZ Tray
   🔗 https://qz.io/download/
   
   • Piliin ang Windows version
   • I-download ang installer
   • I-save sa computer

HAKBANG 2: I-install ang QZ Tray
   
   • I-run ang installer
   • Sundin ang installation instructions
   • I-start ang QZ Tray pagkatapos mag-install

HAKBANG 3: I-configure ang QZ Tray
   
   • Hanapin ang QZ icon sa system tray (baba-kanan)
   • Right-click sa icon
   • I-check ang "Launch on Startup"
   • Verify na "Connected" ang status

HAKBANG 4: I-test ang Connection
   
   • Double-click: OPEN_QZ_TRAY_TEST.bat
   • O kaya buksan: http://localhost/caps2e2/qz-tray-test.html
   • I-click ang "Connect to QZ Tray"
   • Dapat makita: ✅ "Connected successfully"

HAKBANG 5: I-test ang Printer
   
   • I-click ang "Refresh Printers"
   • Piliin ang POS-58 printer
   • I-click ang "Test Print"
   • Dapat mag-print ng test receipt

HAKBANG 6: I-test sa POS System
   
   • Buksan ang POS: http://localhost:3000/POS_convenience
   • Mag-add ng items sa cart
   • Mag-checkout
   • Dapat automatic mag-print ng receipt

HAKBANG 7: I-upload sa Namecheap (Para sa Online)
   
   • Gumamit ng FTP (FileZilla)
   • I-upload ang Api/qz-tray-receipt.php
   • I-upload ang lahat ng Next.js files
   • I-test ang online printing

┌──────────────────────────────────────────────────────────────────┐
│                    KUNG MAY PROBLEMA                             │
└──────────────────────────────────────────────────────────────────┘

PROBLEMA: "QZ Tray is not running"
SOLUSYON:
   1. I-start ang QZ Tray application
   2. Check sa system tray kung may QZ icon
   3. Right-click → verify "Connected"

PROBLEMA: "No printers found"
SOLUSYON:
   1. Check kung naka-install ang printer sa Windows
   2. Buksan: Settings → Printers & scanners
   3. Verify na nandoon ang POS-58
   4. I-click ang "Refresh Printers" sa test page

PROBLEMA: Nag-send ng print job pero walang lumalabas
SOLUSYON:
   1. Check kung may papel ang printer
   2. Check kung online ang printer (walang error light)
   3. Buksan ang Windows Print Queue
   4. Clear stuck print jobs
   5. I-restart ang printer

PROBLEMA: Hindi maka-connect sa QZ Tray
SOLUSYON:
   1. Verify na running ang QZ Tray
   2. I-restart ang QZ Tray
   3. I-restart ang browser
   4. Check kung may firewall na nag-block

PROBLEMA: "QZ Tray not available, falling back to server print"
SOLUSYON:
   • Ito ay normal kung walang QZ Tray
   • Gagamitin ang server printing instead
   • Para gumamit ng QZ Tray, i-install at i-start ito
