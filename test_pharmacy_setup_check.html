‚úÖ How to Add Products to Pharmacy

Follow these steps:

  1. Go to Inventory/Warehouse Module
    • Navigate to: http://localhost:3000/Inventory_Con
    • Click on "Warehouse" tab
  2. Add Products to Warehouse (if empty)
    • Click "Add Product" button
    • Fill in product details (name, barcode, category, SRP, quantity)
    • Select supplier and set expiration date
    • Save the product
  3. Transfer Products to Pharmacy
    • Go to "Inventory Transfer" tab
    • Create a new transfer
    • Source: Warehouse
    • Destination: Pharmacy
    • Select products and quantities to transfer
    • Submit and approve the transfer
  4. Verify Transfer
    • Go to Admin ‚Üí Pharmacy tab
    • Products should now appear!
    • Refresh this test page to verify
